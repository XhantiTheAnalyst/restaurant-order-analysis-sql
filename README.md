# Restaurant Order Analysis in SQL

## Project Overview
This project analyzes restaurant order data using SQL to uncover insights into sales performance, popular menu items, and customer ordering behavior.

## Tools Used
- SQL
- GitHub

## Key Questions
- What are the most popular menu items?
- Which days and times have the highest order volume?
- What is the total and average order value?
- Which items generate the most revenue?

## Dataset
Restaurant order data containing order details, items, prices, and timestamps.
